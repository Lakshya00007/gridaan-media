# Full Stack Media Publishing Platform

This is a React + Vite + Tailwind-style project for a media publishing platform.

## Setup

Install dependencies:

```bash
npm install
```

## Development

Run the local development server:

```bash
npm run dev
```

Open the URL shown in the terminal to view the app in your browser.

## Build

Create a production build:

```bash
npm run build
```

## Preview

Preview the production build locally:

```bash
npm run preview
```

## Notes

- The project builds successfully as of May 21, 2026.
